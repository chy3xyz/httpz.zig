# HTTP/3 (`httpz.h3`)

QUIC transport and HTTP/3 protocol support via ngtcp2 + nghttp3 C bindings.

**Import path:** `httpz.h3`

## Architecture

```
httpz.h3.Client / httpz.h3.Server     ← composed API
httpz.h3.http3.Session                ← HTTP/3 framing (QPACK, headers, data)
httpz.h3.quic.Connection / Listener   ← QUIC transport (UDP, crypto, streams)
```

---

## QUIC Layer (`httpz.h3.quic`)

### `Connection`

```zig
pub const Connection = struct {
    conn: *ngtcp2.ngtcp2_conn,
    socket: posix.fd_t,
    buf: [max_datagram_size]u8,
};
```

| Method | Signature | Description |
|--------|-----------|-------------|
| `deinit` | `fn(self: *Connection) void` | Free QUIC connection + close socket + poison |

### Free Functions

```zig
pub const Error = error{QuicError};

/// Create a QUIC client connection and drive handshake.
pub fn connect(host: []const u8, port: u16) Error!Connection

/// Read a UDP datagram and feed to QUIC engine.
pub fn readPacket(conn: *Connection) Error!void

/// Write pending QUIC packets to UDP socket.
pub fn flushPackets(conn: *Connection) Error!void

/// Get expiry timer in nanoseconds, or null if idle.
pub fn getExpiry(conn: *Connection) ?u64

/// Handle timer expiry (retransmissions, keepalive).
pub fn handleExpiry(conn: *Connection) Error!void
```

### `Listener`

```zig
pub const Listener = struct {
    socket: posix.fd_t,
    connections: std.AutoHashMap([18]u8, *Connection),
    allocator: std.mem.Allocator,

    pub fn init(allocator: std.mem.Allocator, port: u16) Error!Listener
    pub fn deinit(self: *Listener) void
};
```

Binds a UDP socket and routes incoming packets by Connection ID. The `connections` map is populated by `accept()` (pending implementation).

## HTTP/3 Layer (`httpz.h3.http3`)

### `Session`

```zig
pub const Session = struct {
    conn: *nghttp3.nghttp3_conn,
    callbacks: nghttp3.nghttp3_callbacks,
    allocator: std.mem.Allocator,

    pub fn init(allocator: std.mem.Allocator) !Session
    pub fn deinit(self: *Session) void
};
```

Wraps a single HTTP/3 connection. `init()` creates the nghttp3 client connection and installs default callbacks. Callback customization (for server-side `recv_header`/`recv_data` dispatch) is pending.

```zig
pub const Error = error{H3Error};
```

## Usage Example

```zig
const std = @import("std");
const httpz = @import("httpz");
const h3 = httpz.h3;

pub fn main() !void {
    var gpa = std.heap.GeneralPurposeAllocator(.{}){};
    defer _ = gpa.deinit();
    const allocator = gpa.allocator();

    // Client
    var client = try h3.Client.init(allocator, "example.com", 443);
    defer client.deinit();

    // Server
    var server = try h3.Server.init(allocator, 4433);
    defer server.deinit();
}
```

## Dependencies

| `libnghttp3` | ≥ 1.16 | HTTP/3 framing + QPACK |
| `openssl@3` | ≥ 3.0 | TLS 1.3 (QUIC crypto) |

brew install libngtcp2 libnghttp3 openssl@3
